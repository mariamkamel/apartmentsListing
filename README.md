# apartmentsListing
Steps to run:

Backend:

  - cd backend
  - npm install
  - nodemon index.js

In new terminal:

Frontend:

    - cd frontend
    - npm install
    - npm run dev

Mobile:

in your mobile device download expo app https://expo.dev/client  

in a new terminal:

    - cd mobile/apartmentsListingApp
    - npm install
    - npx expo start
    - open the app in your mobile device using the qr code.
